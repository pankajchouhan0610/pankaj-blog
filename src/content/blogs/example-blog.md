Welcome to an example blog post! This is a simple entry meant to show how a blog post looks on the site.

## How this works

- The site uses Markdown files in `src/content/blogs/`.
- Each file should have frontmatter with `title`, `slug`, `description`, and `date`.
- The `slug` is used in the URL (e.g. `/blogs/example-blog`).

## Try it out

Add your own posts by creating new `.md` files with a similar structure. The site will automatically list them under **Recent Writing**.
